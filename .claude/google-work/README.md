# Google Work Account (roman.matsukatov@mindera.com)

В этой папке хранятся **отдельные** OAuth-данные для рабочего Google-аккаунта (не из корня проекта):

- **credentials.json** — OAuth 2.0 Desktop client, созданный в Google Cloud под рабочим аккаунтом (см. инструкцию ниже).
- **google_calendar_token.json** — токен календаря (создаётся при первом входе через браузер).
- **google_drive_token.json** — токен Drive (создаётся при первом входе через браузер).

Все файлы в папке добавлены в `.gitignore`. Пошаговая инструкция по подключению: **`.claude/reference/google-work-account-setup.md`**.
